Audiobookshelf Sync

Connecting to Audiobookshelf

  1. Open your Audiobookshelf instance in a browser
  2. Go to Settings → Users and select your user
  3. Scroll to API Token — copy the token
  4. Enter your ABS server URL (e.g. http://localhost:13378) and paste the token below

What does sync do?

Sync imports your audiobooks and ebooks from Audiobookshelf into Shelf, including metadata and cover art. It also auto-links digital items to matching physical books in your collection.

Hardcover

Connecting to Hardcover

  1. Create a free account at hardcover.app
  2. Go to Account Settings → Hardcover API
  3. Copy the API token shown at the top of the page
  4. Paste it into the field below and click Test

What does Hardcover do?

Hardcover is a modern book tracking platform (like Goodreads). With an API token, Shelf can use Hardcover as a metadata source for richer book data including series info, descriptions, and cover art. Future updates will add library import/export and reading status sync.

The API is completely free. Tokens expire after one year.

Import Library from Hardcover

Reading Status Sync

Pull reading status changes from Hardcover automatically. Status changes made in Shelf are pushed to Hardcover instantly.

Export to Hardcover

Push your Shelf collection to your Hardcover library. Only items with an ISBN can be exported.

Collection Valuation

Getting an ISBNdb API Key

  1. Go to isbndb.com and create an account
  2. Subscribe to a plan that includes API access (Basic or higher)
  3. Navigate to your account dashboard and find the API Keys section
  4. Generate a new key, copy it, and paste it into the field below

About ISBNdb

ISBNdb is a book metadata service that provides publisher list prices (MSRP). These prices are used to estimate the replacement value of your collection for insurance purposes.

API access requires a paid subscription (Basic plan: $14.99/month or $149/year). Valuation results are cached for one year, so you can subscribe monthly, valuate your collection, then cancel — subsequent runs will use cached prices.

Movie Database (TMDb)

Getting a TMDb API Key

  1. Go to themoviedb.org and create a free account
  2. Go to Settings → API in your profile
  3. Request an API key (select "Developer" — it's instant for personal use)
  4. Copy the API Key (v3 auth) value and paste it below

About TMDb

The Movie Database (TMDb) provides movie and TV metadata including poster art. It's used when you scan DVD or Blu-ray UPC barcodes — Shelf looks up the UPC to get the title, then fetches metadata and poster from TMDb.

TMDb's API is free for personal use with generous rate limits.
